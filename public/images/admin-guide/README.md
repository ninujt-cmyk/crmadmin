# Admin Guide Screenshots

This directory contains screenshots for the Admin Capabilities Guide.

## Screenshot Naming Convention

Each screenshot should be named according to the function it represents:

```
{section}_{function}.png
```

Examples:
- `overview_dashboard.png` - Dashboard overview
- `lead-management_all-leads.png` - All Leads page
- `lead-management_upload-leads.png` - Upload Leads page
- `team-management_users.png` - Telecallers/Users page
- `analytics_reports.png` - Reports dashboard
- `system-settings.png` - Settings page
